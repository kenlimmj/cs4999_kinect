    text Class

    Defined in: js/aside.js:1
    Module: quetzalcoatl

    Provides the text class. The text block is located on the left-hand side of the UI, and displays texts, lists, quotes, code, or equations that annotate content in the main block. This class contains helper functions for manipulating the text content in various ways.
