    stabilizer

    (
    • x
    • factor
    )
    Number static

    Defined in js/cursor.js:90

    Rounds a number such that it is a multiple of the provided number. This function comes in handy for stabilizing cursor jitter (by rounding the mapped coordinates).

    Parameters:

    • x Number

      A number to be rounded.

    • factor Number

      A number which represents a multiple to which x will be rounded.

    Returns:

    Number:

    A rounded number.

    Example:

     stabilizer(92,6) = 96
